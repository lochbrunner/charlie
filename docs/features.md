# Features

## Already Supported

* External function calls of the type
 * `void(void)`
 * `void(int)`
 * `void(const char* )`

* Internal function calls of type int with any number of int typed arguments
     
* Operations on Integers
 * `=`, `+`,`-`,`*`,`/`, `%`, `++`, `--`, `>`, `>=`, `<`, `<=`
* Control flow
 * `if`, `while`
